🚀 SimpleAuth V2 – Stronger, Safer, and More Flexible!
This project is a more advanced and secure version of the original SimpleAuth system. It now includes features like email verification, password reset, and improved error handling for a more robust authentication experience. 🎯

On the frontend, I used React, React Router, and Tailwind CSS to build a clean and modern UI. For the backend, Node.js, Express, MySQL, JWT, and bcrypt provide a strong and scalable structure. The overall codebase is now more readable and maintainable. 📚

🆕 What's New in V2:
✅ Email verification (activation link after registration)
✅ "Forgot password" feature with reset email
✅ Strong password validation during reset
✅ Loading states and improved user feedback
✅ Turkish character restrictions for extra input security
✅ More modular and maintainable code structure

🛠️ Core Features:
🔒 User registration and login
🔒 JWT-based token authentication
🔒 Password hashing with bcrypt
🎨 Clean UI with Tailwind CSS
🧭 Page routing with React Router
🔗 RESTful API design
